# Chip-8_Emulator
Chip-8 Emulator with C++ and SDL2 to understand software and emulator architecture

| <kbd> 4 5 6 D </kbd>  | <kbd> Q W E R </kbd>  |
| <kbd> 7 8 9 E </kbd>  | <kbd> A S D F </kbd>  |
| <kbd> A 0 B F </kbd>  | <kbd> Z X C V </kbd>  |

* SDL2
* Made in Windows11
